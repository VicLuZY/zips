# zips

## Quick Start

Run the following commands from the repository root to install dependencies and start frontend + backend together in one terminal:

```bash
npm install
npm run dev
```

If you prefer separate terminals:

```bash
npm run dev --workspace backend
npm run dev --workspace frontend
```

Services:
- Frontend (Vite): http://localhost:5173
- Backend API (Express): http://localhost:8080

Helpful scripts:

```bash
npm run build
npm run lint
```
